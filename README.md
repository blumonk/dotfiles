## Quick Start

Vim 8 is required.

1. Clone this repo.
2. Copy all the files to the home dir `~`
3. Install **vim-plug**:

```
    curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
```

4. Install Plugins:

    Launch `vim` and run `:PlugInstall`
